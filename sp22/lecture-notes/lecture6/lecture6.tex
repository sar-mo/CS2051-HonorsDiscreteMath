\section*{Relations}

    Recall the definition of a relation:
    
    \vspace{1.5mm}
    \textbf{Definition.} A \textit{relation} is a subset of the Cartesian product $A\times B$.
    
    \vspace{1.5mm}
    We denote relations by $\mathcal{R}$. We write $a\mathcal{R} b$ to indicate that $(a,b)\in \mathcal{R}$ (i.e.: in the subset denoted by $\mathcal{R}$). When $A=B$ we say that $\mathcal{R}$ is a relation on $A$.\\
    
    Let $\mathcal{R}_1, \dots, \mathcal{R}_4$ be relations on $A = \{1, 2, 3, 4\}$.
    \begin{itemize}
        \item $\mathcal{R}_1 = \{(a, b) \mid a \le b\}$
        \item $\mathcal{R}_2 = \{(a, b) \mid a = b\}$
        \item $\mathcal{R}_3 = \{(a, b) \mid a+b \le 2022\}$
        \item $\mathcal{R}_4 = \{(a, b) \mid a \text{ divides } b\}$
    \end{itemize}

    So why relations? They are more general and allow us to study more complex mappings between sets where multiple values might be associated with a single input. \\
    
\subsection*{Properties}
    
    \begin{itemize}
        \item \underline{Reflexive:} $(\forall a \in A)(a\mathcal{R}a)$
        \item \underline{Symmetric:} $(\forall a, b \in A)(a\mathcal{R}b \iff b\mathcal{R}a)$
        \item \underline{Antisymmetric:} $(\forall a, b\in A)(a\mathcal{R}b \wedge b\mathcal{R}a \rightarrow a=b)$
        \item \underline{Transitive:} $(\forall a, b, c\in A)(a\mathcal{R}b \wedge b\mathcal{R}c \rightarrow a\mathcal{R}c)$
    \end{itemize}
    
    We checked these properties for $\mathcal{R}_{2}$ and $\mathcal{R}_{3}$. $\mathcal{R}_{2}$ is reflexive, symmetric, and transitive. $\mathcal{R}_{3}$ is symmetric, but \textit{not} transitive (e.g., $2000 \mathcal{R} 0$ and $0 \mathcal{R} 2000$, but $2000 \not\mathcal{R} 2000$).
    
\section*{Functions}
    Last lecture, we informally covered the meaning of a function. With our new knowledge of relations, we now have the tools to formally define them.
    
    \vspace{1.5mm}
    \textbf{Definition.}
    A function is a relation on $A\times B$ such that 
    $$(\forall a \in A)(\exists! b \in B)(a\mathcal{R}b).$$
    
    By the way, the propositional symbol $\exists!$ means ``there exists only one".
    
    Often we are presented with functions describing the unique element $b$ for each $a$. In this case we use $f$ to denote the function and write $f: A \rightarrow B$ or $b=f(a)$. \\
    
    A function $f$ is a way to associate each item in a set to a unique element in another set. \\
    
    Notation:\\
    Usually, we write functions abstractly as $f: A \rightarrow B$.
    
    $A$ is called the domain. $B$ is called codomain. The range of $f$ is the set of all $b\in B$ for which there is at least one $a\in A$ satisfying $f(a)=b$. 
    
    \textbf{Representation using relations:} A function $f: A \rightarrow B$ is a relation $\mathcal{R} \subseteq A \times B$ where for every $a \in A$, there is exactly one pair $(a, b) \in \mathcal{R}$.
    
\subsection*{Classes of Functions}
    \begin{enumerate}
        \item A function is \textit{surjective} (or onto) if every element in $B$ is associated with at least one element in $A$. In other terms, the range is equal to the codomain.
        \item A function is \textit{injective} (or one-to-one) if no two distinct elements $a_1, a_2 \in A$ map to the same element in $B$. In other terms, $f(a_1) = f(a_2) \rightarrow a_1 = a_2$.
        \item A function is \textit{bijective} if it is surjective and injective.
    \end{enumerate}
    
    Examples:
    What are the domain and codomain of the following functions? Also determine whether the following functions are surjective, injective, or both (bijective).
    \begin{itemize}
        \item $f$ assigns to each student in the class their height in cm.
        \item $f(x) = x+1$ from $\mathbb{N}$ to itself, then from $\mathbb{Z}$ to itself.
        \item $f$ assigns to each bit string of length two or more its last two bits.
        \item $f$ assigns to each real number the largest integer less than or equal to the number.
    \end{itemize}
    
    Solutions:
    \begin{itemize}
        \item Domain: Students, Codomain: $\mathbb{R}^{+}$. Neither injective nor surjective.
        \item This function is only injective when mapping from $\mathbb{N}$ to itself, as $f(x) = 1$ cannot be reached by any $x \in \mathbb{N}$ (assuming $0 \notin \mathbb{N}$). However, when mapping from $\mathbb{Z}$ to itself, this function becomes bijective.
        \item D: bitstrings of length $\ge 2$, C: bitstrings of length 2. Surjective.
        \item $\mathbb{R} \rightarrow \mathbb{Z}$. Surjective.
    \end{itemize}
    
    In the language of relations:
    \begin{itemize}
        \item \textbf{One-to-one (Injective):} If $(a_1, b) \in \mathcal{R}$ and $(a_2, b) \in \mathcal{R}$, then $a_1 = a_2$.
        \item \textbf{Onto (Surjective):} For every $b \in B$, there exists at least one $a \in A$ such that $(a, b) \in \mathcal{R}$.
    \end{itemize}
    
    The general strategy to prove a function is one-to-one is to assume the outputs are equal and show the inputs must be equal:
    $$f\text{ is one-to-one if } f(x) = f(y) \rightarrow x = y$$
    
    And onto:
    $$f\text{ is onto if } (\forall b\in B)(\exists a\in A) (f(a)=b)$$
    
    \textbf{Example Proofs:}
    \begin{itemize}
        \item \textit{Prove $f(x) = 2x$ from $\mathbb{R}$ to $\mathbb{R}$ is one-to-one:} Assume $f(x) = f(y)$. Then $2x = 2y$. Dividing by 2, we get $x = y$. Thus, $f$ is one-to-one.
        \item \textit{Prove $f(x) = 2x$ from $\mathbb{R}$ to $\mathbb{R}$ is onto:} Let $b \in \mathbb{R}$. We must find an $a \in \mathbb{R}$ such that $f(a) = b$. Let $a = \frac{b}{2}$. Since $b \in \mathbb{R}$, $a \in \mathbb{R}$. Then $f(a) = 2(\frac{b}{2}) = b$. Thus, $f$ is onto.
    \end{itemize}

\subsection*{Inverse Functions}
    If a function $f: A \rightarrow B$ is a bijection, its \textit{inverse} relation $f^{-1} = \{(b, a) \mid (a, b) \in f\}$ is also a valid function mapping from $B$ to $A$. We write $f^{-1}(b) = a \iff f(a) = b$.

\section*{Equivalence Relations}
    
    \textbf{Definition:} A relation is said to be an equivalence relation if it is reflexive, symmetric, and transitive. 

    \vspace{1.5mm}
    Example: Show that the relation on the real numbers defined as $a\mathcal{R}b \leftrightarrow a-b\in \mathbb{Z}$ is an equivalence relation.
    
    \vspace{1.5mm}
    Solution:
    \begin{enumerate}
        \item \textbf{Reflexive:} Let $a \in \mathbb{R}$. Then $a - a = 0$. Since $0 \in \mathbb{Z}$, $a \mathcal{R} a$.
    \item \textbf{Symmetric:} Assume $a \mathcal{R} b$. Then $a - b = k$ for some $k \in \mathbb{Z}$. Then $b - a = -(a - b) = -k$. Since $k \in \mathbb{Z}$, $-k \in \mathbb{Z}$, so $b \mathcal{R} a$.
    \item \textbf{Transitive:} Assume $a \mathcal{R} b$ and $b \mathcal{R} c$. We wish to show $a \mathcal{R} c$. We know $a - b = d$ for some $d \in \mathbb{Z}$, and $b - c = e$ for some $e \in \mathbb{Z}$. Adding these equations yields $(a - b) + (b - c) = d + e$, which simplifies to $a - c = d + e$. Since integers are closed under addition, $d + e \in \mathbb{Z}$. Thus, $a \mathcal{R} c$.
    \end{enumerate}
    
    \textbf{Definition.} Given a set $A$ and an equivalence relation on it, the equivalence classes are the sets
    $$[a]_{\mathcal{R}}=\{x\in A~|~a\mathcal{R} x\}$$
    
    \textbf{Theorem.} Let $A$ be a set and $\mathcal{R}$ an equivalence relation on it. The following are equivalent:
    \begin{itemize}
        \item[(i)] $a\mathcal{R}b$.
        \item[(ii)] $[a]_{\mathcal{R}}=[b]_{\mathcal{R}}$.
        \item[(iii)] $[a]_{\mathcal{R}}\cap [b]_{\mathcal{R}}\neq \emptyset$.
    \end{itemize}
    
    \textbf{Proof.} 
    \begin{itemize}
        \item (i) to (ii): Suppose $a \mathcal{R} b$. Let $c \in [a]_{\mathcal{R}}$. Then $a \mathcal{R} c$. Since $\mathcal{R}$ is symmetric, $a \mathcal{R} b$ implies $b \mathcal{R} a$. By transitivity on $b \mathcal{R} a$ and $a \mathcal{R} c$, we have $b \mathcal{R} c$. Thus $c \in [b]_{\mathcal{R}}$, showing $[a]_{\mathcal{R}} \subseteq [b]_{\mathcal{R}}$. By identical logic in reverse, $[b]_{\mathcal{R}} \subseteq [a]_{\mathcal{R}}$. Therefore, $[a]_{\mathcal{R}} = [b]_{\mathcal{R}}$.
        \item (ii) to (iii): Suppose $[a]_{\mathcal{R}}=[b]_{\mathcal{R}}$. By reflexivity, $a \mathcal{R} a$, which means $a \in [a]_{\mathcal{R}}$. Since the sets are equal, $a \in [b]_{\mathcal{R}}$. Therefore, $a \in [a]_{\mathcal{R}}\cap [b]_{\mathcal{R}}$, so $[a]_{\mathcal{R}}\cap [b]_{\mathcal{R}}\neq \emptyset$.
        \item (iii) to (i): Assume $[a]_{\mathcal{R}} \cap [b]_{\mathcal{R}} \ne \emptyset$. Then there exists some $c \in [a]_{\mathcal{R}} \cap [b]_{\mathcal{R}}$. This means $a \mathcal{R} c$ and $b \mathcal{R} c$. By symmetry, we have $c \mathcal{R} b$. By transitivity on $a \mathcal{R} c$ and $c \mathcal{R} b$, we conclude that $a \mathcal{R} b$, as desired.
    \end{itemize}
    
    \vspace{1.5mm}
    \textbf{Definition.} A collection of non-empty subsets $P_1, P_2, \dots, P_k$ is a \textit{partition} of a set $A$ iff $\bigcup_{i = 1}^{k} P_{i} = A$, and the sets are pairwise disjoint (i.e., $P_i \cap P_j = \emptyset$ for all $i \neq j$).
    
    Why is every element in a class? Answer: Because of the reflexive property ($a \in [a]_{\mathcal{R}}$). Why do equivalence classes form a partition? Because of facts (ii) and (iii) of the Theorem; if two classes overlap at all, they must be the exact same class, meaning distinct classes are completely disjoint.

\section*{Post Lecture}

\subsection*{Question 1}
    Al and Bob play a game. They have the numbers $1,2,\dots, 9$ written on cards face up. Players alternate taking any card. The first player to have exactly 3 cards whose sum is 15 wins. Determine which player, if any, has a winning strategy.
    
    \vspace{1.5mm}{\small \textit{Hint}: The game `Tic-Tac-Toe' is known to have no winning strategy for either player. Furthermore, I strongly encourage you to play this game a few times with a friend, and see what strategies work.}

\subsection*{Solution}
    Either by playing this game yourself, or by doing the calculations yourself, you may have noticed there are eight possible ways to win this game. Furthermore, you can create a bijection mapping these possibilities to the magic square shown below:
    \begin{center}
        \begin{tabular}{c|c|c}
            4 & 9 & 2 \\
             \hline
            3 & 5 & 7 \\
             \hline
            8 & 1 & 6
        \end{tabular}
    \end{center}
    Since Al and Bob are alternating turns, they are essentially playing Tic-Tac-Toe. As Tic-Tac-Toe is known to have no winning strategy for either player, we conclude that there is no winning strategy for Al or Bob.

\subsection*{Question 2}
    Let $f(x) = x - 1$ for all $x \in [1, \infty)$. Show that $f: [1, \infty) \rightarrow [0, \infty)$ is an injection. Then show that $\text{Rng}(f) = [0, \infty)$. Conclude that $f$ is a bijection from $[1, \infty)$ to $[0, \infty)$.

\subsection*{Solution}
    Suppose $x \in [1, \infty)$ and $y = f(x)$. Since $1 \le x < \infty$, we have $1 - 1 \le x - 1 < \infty - 1$, so $0 \le x - 1 < \infty$, so $f(x) \in [0, \infty)$. Also, since $y = f(x) = x - 1$, we have $x = y + 1$. Thus $f: [1, \infty) \rightarrow [0, \infty)$.
    
    \vspace{1.5mm}
    Now suppose $f(x_{1}) = f(x_{2})$ for $x_{1}, x_{2} \in \text{Dom}(f)$. Then $x_{1} - 1 = x_{2} - 1$, so $x_{1} = x_{2}$. Thus $f$ is injective.
        
    \vspace{1.5mm} To show that $\text{Rng}(f) = [0, \infty)$, it remains only to show that each $y$ in $[0, \infty)$ belongs to $\text{Rng}(f)$. Let $y \in [0, \infty)$. We wish to show that there exists $x \in [1, \infty)$ such that $f(x) = y$. Let $x = y + 1$. Then $y = x - 1$. Hence, once we have shown that $x \in [1, \infty)$, we will have that $f(x) = x - 1 = y$. Now $0 \le y < \infty$, so $0 + 1 \le y + 1 < \infty + 1$, so $1 \le y + 1 < \infty$, so $1 \le x < \infty$, so $x \in [1, \infty)$. Thus $\text{Rng}(f) = [0, \infty)$, so $f$ is surjective, and we can conclude that $f$ is a bijection from $[1, \infty)$ to $[0, \infty)$.
